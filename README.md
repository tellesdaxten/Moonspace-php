# Moonspace

Moonspace é uma rede social criada em 2022 que tem como objetivo conectar jovens do mundo todo com os mesmos interesses, permitindo que você não se torne um produto das redes sociais, mas sim, um protagonista do seu perfil.

Este repositório é da versão que teve sua última atualização no ano de 2023. A partir de agora, a Moonspace será reescrita sob novas liguagens de programação e terá outras funcionalidades.
